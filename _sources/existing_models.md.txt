# Existing flight models

Flight models based on
- FW2011 (to be added, matlab based)
- [Seaglider basestation](https://github.com/iop-apl-uw/basestation3/blob/master/FlightModel.py) designed to run on Seaglider data, and
- from Lucas Merckelbach's [gliderflight](https://gliderflight.readthedocs.io/en/latest/using_gliderflight.html) designed to run on Slocum data.
